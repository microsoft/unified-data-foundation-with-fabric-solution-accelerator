# User Sample Test Questions - Microsoft Fabric Data Agent

## Overview

This document contains a collection of test questions designed to validate the Microsoft Fabric Data Agent's capabilities across different scenarios and complexity levels. These questions cover data exploration, business analytics, cross-domain analysis, and performance optimization within the Gold tier data lake.
